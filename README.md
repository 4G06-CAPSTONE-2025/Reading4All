# Reading4All

Developer Names: Nawaal Fatima, Dhruv Sardana, Fiza Sehar, Moly Mikhail, Casey Francine Bulaclac  

Date of project start: Sept 15 2025

This project is an AI-powered accessibility tool that automatically generates descriptive alt-text for technical diagrams, supporting AODA and WCAG 2.1 compliance. 

## Key Features  

### Alt-Text Generation  
Automatically generates clear, descriptive alt-text for technical and educational diagrams using vision-language models.

### Dataset & Training Tools  
Includes preprocessing and annotation tools for building and evaluating training datasets.

### Inference Module  
Provides fast alt-text generation for new images with configurable output settings.

### Frontend Interface  
Offers a simple UI for uploading diagrams and viewing or editing generated descriptions.
