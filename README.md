# Reading4All

> An AI-powered accessibility tool that automatically generates descriptive alt-text for technical diagrams — supporting AODA and WCAG 2.1 compliance in real time.

**Developers:** Nawaal Fatima, Dhruv Sardana, Fiza Sehar, Moly Mikhail, Casey Francine Bulaclac  
**Project Start:** September 15, 2025  
**Status:** Completed as of April 2026 (Capstone Submission)

---

## What is Reading4All?

Reading4All is a capstone project built to address a real accessibility gap: technical and educational diagrams are frequently published without meaningful alt-text, making them inaccessible to users who rely on screen readers or assistive technologies.

This tool allows users to upload a diagram and instantly receive a clear, descriptive alt-text generated by a custom fine-tuned vision-language model — no manual writing required. The output is designed to meet AODA and WCAG 2.1 accessibility standards.

---

## Key Features

- **Instant Alt-Text Generation** — Upload any technical or educational diagram and receive a descriptive alt-text in seconds, powered by a custom fine-tuned vision-language model.
- **Accessibility Compliant** — Output is aligned with AODA and WCAG 2.1 guidelines, making it suitable for academic, professional, and public-facing content.
- **Dataset & Training Tools** — Includes preprocessing and annotation utilities for building and evaluating training datasets.
- **Inference Module** — Fast, configurable alt-text generation pipeline for new images.
- **Simple Frontend Interface** — A clean UI for uploading diagrams and viewing generated descriptions instantly.
