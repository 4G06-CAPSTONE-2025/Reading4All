# docs/Design/

This folder contains the software design documentation for Reading4All, covering both the high-level system architecture and the detailed module-level design.

---

## Subfolders

| Folder | Document | Description |
|---|---|---|
| `SoftArchitecture/` | Module Guide (MG) | High-level software architecture — system decomposition, module hierarchy, and Figma UI mockups |
| `SoftDetailedDes/` | Module Interface Specification (MIS) | Detailed design — module interfaces, access routines, and state variables |

---

## Notes

- Both subfolders contain a `.tex` source, compiled `.pdf`, and a `Makefile`
- `SoftArchitecture/` also contains Figma UI screenshots for all major pages (homepage, login, upload, history, success/failure states)
- `SoftDetailedDes/MIS.pdf` is the primary reference for understanding how individual modules communicate
