# docs/Design/SoftDetailedDes/

This folder contains the Module Interface Specification (MIS) for Reading4All — the detailed design document describing each module's interface, access routines, inputs, outputs, and state.

---

## Files

| File | Description |
|---|---|
| `MIS.pdf` | Compiled Module Interface Specification — primary reading |
| `MIS.tex` | LaTeX source for the MIS |
| `Makefile` | Build script — run `make` to recompile the PDF |

---

## Notes

- The MIS is the detailed counterpart to the Module Guide in `SoftArchitecture/`
- It specifies how each module is accessed, what it expects as input, and what it returns — useful for understanding internal API contracts
