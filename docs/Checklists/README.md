# docs/Checklists/

This folder contains the capstone milestone checklists used to verify that each deliverable meets course requirements before submission. Each checklist exists as both a LaTeX source (`.tex`) and compiled PDF (`.pdf`).

---

## Files

| Checklist | Description |
|---|---|
| `Code-Checklist` | Checklist for code quality and standards |
| `DevPlan-Checklist` | Checklist for the Development Plan deliverable |
| `FinalDoc-Checklist` | Checklist for final documentation |
| `GettingStarted-Checklist` | Checklist for initial project setup deliverable |
| `MG-Checklist` | Checklist for the Module Guide |
| `MIS-Checklist` | Checklist for the Module Interface Specification |
| `POC-Checklist` | Checklist for the Proof of Concept demo |
| `ProbState-Checklist` | Checklist for the Problem Statement deliverable |
| `SRS-Checklist` | Checklist for the SRS (general) |
| `SRS-SciComp-Checklist` | Checklist for the SRS using the Scientific Computing template |
| `VnV-Checklist` | Checklist for the VnV Plan and Report |
| `Writing-Checklist` | Checklist for general writing quality across deliverables |

---

## Notes

- Each checklist has a `.tex` source and a compiled `.pdf`
- These were used internally to self-assess deliverables before submission — they are not the deliverables themselves
