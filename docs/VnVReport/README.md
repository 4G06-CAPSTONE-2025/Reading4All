# docs/VnVReport/

This folder contains the Verification and Validation Report (Test Report) for Reading4All — documenting the results of executing the test plan defined in `VnVPlan/`.

---

## Files

| File | Description |
|---|---|
| `VnVReport.pdf` | Compiled VnV Report — primary reading |
| `VnVReport.tex` | LaTeX source |
| `Makefile` | Build script — run `make` to recompile the PDF |
| `BackendCoverage.png` | Screenshot of backend test coverage |
| `BackendCoverage_Updated.png` | Updated backend coverage screenshot |
| `FrontendCoverage.png` | Screenshot of frontend test coverage |
| `R4A_VnVRep_autoTest.png` | Screenshot of automated test run results |

---

## Notes

- Coverage screenshots are embedded in the report as evidence of test execution
- See `VnVPlan/` for the test plan that this report executes against
