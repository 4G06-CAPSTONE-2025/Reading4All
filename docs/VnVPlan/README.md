# docs/VnVPlan/

This folder contains the Verification and Validation Plan for Reading4All — a capstone deliverable describing the testing strategy, test cases, and evaluation approach used to verify the system meets its requirements.

---

## Files

| File | Description |
|---|---|
| `VnVPlan.pdf` | Compiled VnV Plan — primary reading |
| `VnVPlan.tex` | LaTeX source |
| `Makefile` | Build script — run `make` to recompile the PDF |

---

## Notes

- The VnV Plan defines both automated and manual test cases across frontend, backend, and AI components
- See `VnVReport/` for the results of executing this plan
