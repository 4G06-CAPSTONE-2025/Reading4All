# docs/SRS-Volere/

This folder contains the Software Requirements Specification (SRS) for Reading4All, written using the Volere requirements template.

---

## Files

| File | Description |
|---|---|
| `SRS.pdf` | Compiled SRS — primary reading |
| `SRS.tex` | LaTeX source |
| `images/` | Diagrams and figures referenced in the SRS |

---

## Notes

- The Volere template was selected from the three SRS options available in the capstone course
- The SRS defines all functional and non-functional requirements for the system, including AODA and WCAG 2.1 accessibility compliance requirements
